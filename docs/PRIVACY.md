# LexiTranslate - 划词翻译助手 隐私说明

最后更新日期：2026-04-26

## 我们收集什么

本扩展**不**要求您登录账号，**不**收集您的浏览历史或书签。

本扩展仅在您**主动选中**网页上的文本并触发翻译时，将该段文本发送到您所选择的第三方翻译服务（MyMemory 和/或 LibreTranslate 公共接口），以获取翻译结果。

## 本地存储

我们使用 Chrome 提供的 `storage` 功能，在您的浏览器本地保存偏好设置，例如：目标语言、主题、翻译引擎、字号、快捷键组合等。这些数据保存在您的设备上，不由我们运营的服务器收集。

## 第三方服务

翻译功能依赖第三方 API。请求将发送至：

- MyMemory（`api.mymemory.translated.net`）
- LibreTranslate 公共实例（`libretranslate.de`）

具体数据处理规则请参阅各服务提供方的条款与隐私政策。

## 联系我们

如有隐私相关问题，请通过您发布扩展时填写的支持邮箱或项目仓库 Issue 与开发者联系。

## 简短说明（约 132 字以内）

划词即可翻译外文网页：支持多翻译引擎与自动回退、朗读、复制译文与快捷键，可自选目标语言与主题。

## 详细说明

**LexiTranslate - 划词翻译助手** 帮助你在浏览网页时快速理解外文内容。

**你能用它做什么**

- 选中页面上的词句，一键翻译并在浮窗中查看原文与译文  
- 可选翻译引擎（MyMemory、LibreTranslate），主服务异常时自动尝试备用引擎  
- 朗读原文或译文，方便听读学习  
- 一键复制译文  
- 在设置中调整目标语言、主题与页面内快捷键  
- 支持通过扩展命令快捷键触发翻译（可在 `chrome://extensions/shortcuts` 中修改，避免与系统或其他扩展冲突）

**适用场景**

阅读新闻、技术文档、论文与多语言资料时的即时辅助工具。

**说明**

翻译由第三方在线服务完成，需要网络连接；翻译质量受所选服务影响，仅供参考。

## 单一用途说明（审核常用）

本扩展仅用于：在用户主动选中文本的前提下，提供网页划词翻译、结果展示、朗读与复制，以及保存用户本地偏好设置（语言、主题、快捷键等）。不用于广告、挖矿或收集浏览历史。

## 权限与主机权限说明（填「权限理由」时可参考）

| 权限 / 域名 | 用途 |
|------------|------|
| `storage` | 将您的设置（目标语言、主题、翻译引擎、字号、快捷键组合等）保存在本地，下次打开浏览器仍有效。 |
| `activeTab` | 在您使用扩展命令快捷键时，向当前活动标签页发送翻译指令；不读取无关标签页。 |
| `https://api.mymemory.translated.net/*` | 调用 MyMemory 翻译接口，将您选中的文本发送至该服务并返回译文。 |
| `https://libretranslate.de/*` | 调用 LibreTranslate 公共实例翻译接口，将您选中的文本发送至该服务并返回译文。 |

# LexiTranslate - 划词翻译助手

LexiTranslate 是一款为中文用户设计的 Chrome 划词翻译插件。  
在网页中选中文字后，即可快速完成翻译、朗读与复制，适合阅读外文资讯、技术文档和学习资料。

---

## 为什么选择 LexiTranslate

- **划词即翻译**：无需切换页面，阅读不中断
- **双引擎保障**：支持 `MyMemory` 与 `LibreTranslate`
- **自动回退机制**：主引擎不可用时自动切换备用引擎
- **多语言支持**：中文、英语、日语、韩语、法语、西班牙语
- **体验友好**：原文译文并排展示，支持朗读和复制
- **可个性化设置**：目标语言、主题、快捷键均可配置

---

## 30 秒快速开始（开发者模式安装）

1. 打开 Chrome，访问：`chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本项目根目录
5. 安装完成后，打开任意网页，选中一段文字即可使用

---

## 如何使用

1. 在任意网页中选中一段文字
2. 点击浮动工具条中的“翻译”按钮（或使用快捷键）
3. 在翻译面板中查看结果
4. 可继续进行：
   - 复制译文
   - 朗读原文/译文
   - 调整显示字号
5. 如需个性化设置，打开扩展设置页进行调整

默认快捷键：
- Windows / Linux：`Ctrl + Shift + T`
- macOS：`Command + Shift + T`

---

## 功能一览

- 划词后显示快捷工具条（翻译 / 朗读）
- 最多支持 500 字符即时翻译
- 翻译失败自动回退到备用引擎
- 原文与译文双栏展示
- 支持译文复制与双语朗读
- 支持目标语言、主题、快捷键自定义

## 常见问题（FAQ）

### 1) 为什么没有看到划词工具条？
- 请确认你选中了网页中的“可选择文本”
- 某些特殊网页（如浏览器内置页、部分受限页面）不允许注入脚本
- 可刷新页面后重试

### 2) 翻译失败怎么办？
- 先检查网络连接是否正常
- 更换翻译引擎后重试
- 稍后再试，可能是第三方翻译服务暂时不可用

### 3) 快捷键无效怎么办？
- 可能与其他扩展或系统快捷键冲突
- 打开 `chrome://extensions/shortcuts` 修改快捷键

### 4) 支持翻译多长文本？
- 单次最多支持约 **500 字符**（为保证速度与稳定性）

---

## 隐私说明

- 插件只处理你主动选中的文本
- 翻译请求会发送到你选择的第三方翻译服务
- 插件不采集浏览历史，不要求登录账号
- 本地仅保存个性化设置（如主题、目标语言、快捷键）
